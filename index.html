About

PianWHOA! is a virtual piano with a tutor mode programmed in C++ using QT Creator. This project was created by Emilie Menard Barnard(@embarnard), Brittany Berin(@bmbritt10), Bernice Chen, and Kristen Morse for CS 48 at UC Santa Barbara in Winter 2011.

Usage

PianWHOA! features two main types of play: Tutor Mode and Free Play. In Free Play, use your keyboard to play to your heart's desire. If you'd like to learn how to play the piano, use the Tutor Mode. Tutor Mode currently supports five songs. Press the keys as they light up to play the tune. If you need more help, be sure to listen to the song in full using the button in Tutor Mode.

Download Instructions

Download the .zip file and save to your computer. Consult the QT downloads page and download the appropriate version of the QT library (this is necessary for PianWHOA! to run). Then open the Piano1.app file to start playing!
